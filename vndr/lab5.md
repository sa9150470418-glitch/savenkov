# Лабораторная работа №5. Тестирование качества монитора

**Цель:** диагностика монитора и выбор оптимальных параметров цветопередачи.

---

## 1. Характеристики монитора (пример)

| Параметр | Значение |
|----------|----------|
| Модель | ASUS ProArt PA278CV |
| Разрешение | 2560 × 1440 (WQHD) |
| Соотношение сторон | 16:9 |
| Разъем подключения | HDMI 1.4 |
| Диагональ | 27″ |
| Цветов | 16.7 млн |
| Плотность пикселей | 109 PPI |

## 2. Назначение тестов

| Тест | Что проверяет |
|------|---------------|
| Scale | Равномерность градиентов, битые пиксели |
| Gamma | Гамма-кривую (передачу полутонов) |
| Contrast | Детализацию в тенях и светах |

---

## 3. Результаты тестирования

### 3.1 Scale (градиенты)
- R, G, B: плавные переходы, banding слабый, битых пикселей нет.
- White: градиент чистый.
- Углы обзора: при вертикальном отклонении цвета темнеют.

### 3.2 Gamma (белый канал)

| мм | Gamma |
|----|-------|
| 40 | 0.88 |
| 80 | 0.96 |
| 120 | 1.03 |
| 160 | 1.12 |
| 200 | 1.21 |
| 240 | 1.38 |
| 280 | 1.52 |
| 320 | 1.64 |
| 360 | 1.79 |

**Вывод:** оптимум ~1.38–1.52, полутона чуть завышены.

### 3.3 Contrast
- Градации различимы до 99% белого и до 2% чёрного.
- Оптимальный шаг насыщенности — 19%.

### 3.4 Pixel Persistence
- Шлейф <2.2 мм, инерционность средняя.

### 3.5 Printer Test
- С белым сливаются оттенки 1–3% яркости.
- С чёрным сливаются оттенки 2–4% яркости.

## 4. Контрольные вопросы (кратко)

### Виды мониторов
- ЭЛТ — устаревшие, тяжёлые.
- ЖК (LCD) — TN (быстрый), IPS (цвета), VA (контраст).
- OLED — идеальный чёрный, выгорание.
- Плазма — для ТВ, редки в ПК.

### Конструкция ЖК (снизу вверх)
1. Подсветка (LED)
2. Рассеиватель
3. Поляризатор
4. TFT-подложка
5. Жидкие кристаллы
6. Светофильтр (RGB)
7. Второй поляризатор
8. Защитное стекло
